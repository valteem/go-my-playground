Image name is assigned as follows:

|----------------|------------------------------|
| Command        | Name                         |
|--------------|--------------------------------|
| docker build   | defined by -t (tag) option   |
|----------------|------------------------------|
| docker compose | <folder_name>-<service_name> |
|----------------|------------------------------|
| docker-compose | <folder_name>_<service_name> |
|----------------|------------------------------|
